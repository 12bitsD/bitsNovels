# 复古理科与数字羊皮纸 (Retro-Science & Digital Parchment)

> AI 辅助长篇小说创作平台 — 视觉与交互设计规范

------

## 1. 核心理念：降低认知负荷与眼部疲劳

作为一款写作软件，**长时间凝视**是最高频的场景。

- **低对比度原则**：摒弃纯白（#FFFFFF）和纯黑（#000000）的搭配。所有的背景色都带有一点点暖色调（像陈年纸张），所有的文字色都带有一点点棕灰色调（像褪色的墨水）。这大大减少了屏幕眩光。
- **空间收纳**：采用三栏式布局，且左右两侧（大纲和知识库）均可折叠。所有的次要操作（如项目设置、导出）全部收纳进 Modal 弹窗或下拉菜单中，确保编辑区（EditorArea）永远是视觉绝对中心。

------

## 2. 色彩系统 (Color Palette)：羊皮纸与琥珀光

### 环境基底 (Parchment Series)

| Token | 色值 | 用途 |
|---|---|---|
| `--color-parchment-base` | `#fdfbf7` | 主编辑区背景，极浅的米白色，模拟新纸张 |
| `--color-parchment-sidebar` | `#F5F2EB` | 侧边栏背景，比主区稍深，形成物理层级感 |
| `--color-parchment-border` | `#E5E0D8` | 极其克制的边框色，像纸张折叠的阴影 |

### 墨水系统 (Ink Series)

| Token | 色值 | 用途 |
|---|---|---|
| `--color-ink-primary` | `#3E3832` | 正文颜色，深棕黑，比纯黑温和 |
| `--color-ink-secondary` | `#57534E` | 次要文本 |
| `--color-ink-muted` | `#8A7E71` | 说明、图标等辅助信息 |

### 点缀色 (Amber Glow)

| Token | 色值 | 用途 |
|---|---|---|
| `--color-amber-glow` | `#B45309` | 温暖的琥珀/铜橙色，替代"主色蓝"。用于激活 Tab、Hover 状态、AI 功能高亮，如同黄铜笔尖划过羊皮纸的反光，带有"蒸汽朋克"或"复古机械"的味道 |

------

## 3. 排版与字体 (Typography)：人文与理性的交织

长篇小说创作既是文学（人文），其背后的世界观和逻辑（知识库）又是高度结构化的（理性）。

| 场景 | 字体 | 说明 |
|---|---|---|
| 正文编辑区 | `--font-serif` (Noto Serif SC) | 强制使用衬线体，天然文学感，沉浸式阅读/书写；CSS 强制首行缩进 `text-indent: 2em` |
| 侧边栏与数据面板 | 系统默认无衬线体 | 确保界面信息清晰易读 |
| 标签与数字 | `--font-mono` (JetBrains Mono 等宽) | 精密仪器仪表感，与知识库中"神枢图鉴"、"实体拓扑"等科幻/奇幻命名相得益彰 |

------

## 4. 布局结构：三栏式沉浸布局

```
+------------------+------------------------+------------------+
|   左侧面板        |      编辑区             |    右侧面板       |
|   (章节树)        |   (EditorArea)          |    (知识库)       |
|                  |                        |                  |
|  可折叠           |   内容居中              |   可折叠          |
|  700~800px 最大   |   降低视觉重心           |                  |
+------------------+------------------------+------------------+
```

- **左右面板均可折叠/展开**（快捷键或点击边缘拖柄）
- **编辑区自动扩展**至全宽
- **编辑区内容居中**，最大宽度约 700~800px，两侧留白

------

## 5. 视觉隐喻 (Visual Metaphors)

### AI 的具象化

AI 不是冰冷的机器人，而是被隐喻为**"灵感火花"**：

- AI 工具栏使用 **Sparkles ✨**、**Wand2 🪄**（魔法棒）图标
- 琥珀色高亮
- AI 的介入感觉像是一种"炼金术"或"魔法辅助"，减轻机器自动生成的工业感

### 拓扑图的网格背景

RelationshipGraphModal（实体拓扑投影）中：

- CSS 绘制细微的**点状网格背景**
- 配合连线和圆点
- 营造"在战术沙盘或实验桌上推演人物关系"的沉浸感

## 7. 设计目标总结

> **让界面消失。**

通过低对比度的暖色调、精致的排版比例和克制的交互动效，创造一个像**"实木书桌与泛黄羊皮纸"**一样的环境。它不喧宾夺主，但当你需要管理复杂的设定（大纲、知识库）时，它又能像精密的仪器一样为你提供结构化的视图。

这种**"复古文学感"**与**"现代数据管理"**的碰撞，是最适合 AI 辅助长篇小说创作平台的视觉基调。
