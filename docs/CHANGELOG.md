# CHANGELOG · 变更记录

> 每次提交后，在顶部追加记录。不写"修了个 bug"，写"哪条需求变了导致这个改动"。

格式：
```
## YYYY-MM-DD · [commit hash] · [简短描述]

**需求关联**：US-X.X 或 D-XXX
**变更文件**：列出变更的文件
**变更摘要**：一句话描述
```
