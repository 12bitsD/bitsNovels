# CONTEXT · 项目上下文

> AI 辅助长篇小说创作平台

------

## 产品一句话定位

bitsNovels 是一个**融合了 AI 知识管理的专业长篇小说写作工作室**，让作者在同一个界面里完成从创作到设定管理的全部工作。

核心差异化：普通写作工具只能"写"，bitsNovels 能帮你"记住"——AI 自动识别正文中的人物、地点、道具、势力、伏笔，构建可追踪、可视化、可导出的知识库。

------

## 目标用户

- 每天花 **8 小时以上**创作长篇小说的作者
- 需要管理复杂世界观和人物关系（50+ 角色、跨卷伏笔）的作者
- 希望 AI 辅助但不想要"AI 替我写"的控制权焦虑的作者

## 技术栈方向（参考）

| 层级 | 候选方案 | 说明 |
|---|---|---|
| 前端框架 | React + TypeScript | 强类型，适合复杂状态管理 |
| 编辑器 | Slate / ProseMirror | 富文本定制 |

| AI | OpenAI GPT API 或等效 | Parser + 续写 + 润色 |

> 技术栈决策尚未最终确认，Phase 0 启动前需在 `DECISIONS.md` 中记录 Why。
