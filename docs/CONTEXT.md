# bitsNovels · 项目上下文

> AI 辅助长篇小说创作平台 | 2026-03-20 | 阶段：规划（Sprint 0）

------

## 产品一句话定位

bitsNovels 是**融合了 AI 知识管理的专业长篇小说写作工作室**，让作者在同一个界面里完成创作到设定管理的全部工作。

核心差异：AI 自动识别正文中的人物、地点、道具、势力、伏笔，构建可追踪、可视化的知识库。

## 目标用户

- 每天花 **8 小时以上**创作长篇小说的作者
- 需要管理复杂世界观和人物关系（50+ 角色、跨卷伏笔）的作者
- 希望 AI 辅助但不想让 AI 接管写作控制权的作者

## 当前阶段

**Sprint 0 · 规划阶段**

当前任务：需求拆分 + 技术选型决策（Phase 0 启动前需完成）

## 开发方式

**TDD + 敏捷迭代**，2 周一个 Sprint。
FE Agent 和 BE Agent 并行开发，通过 EPIC/N/context.md 共享数据结构与接口约定。

## Epic 依赖顺序

```
EPIC1（项目管理）     → 无前置依赖，最先做
EPIC3（编辑器）       → 项目结构是编辑器的基础
EPIC2（知识库）       → 依赖 EPIC3 的章节结构
EPIC4（AI 写作）     → 依赖 EPIC2 知识库 + EPIC3 编辑器
EPIC5（导出备份）     → 依赖 EPIC3 编辑器
EPIC6（系统设置）    → 无业务依赖，最后做
```

## 技术栈（Sprint 0 决策）
