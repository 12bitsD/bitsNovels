# bitsNovels · 数字羊皮纸

> AI 辅助长篇小说创作平台

------

## 这是一款什么产品

bitsNovels 是一个**长篇小说写作工具**，专为每天可能要花 8 小时以上写小说的人设计。

它的核心功能是让你在一个界面里完成所有创作工作：

- **写**：自带专业富文本编辑器，支持加粗、斜体、章节管理
- **理**：AI 自动从你写的内容里识别角色、地点、道具、势力、伏笔，构建"知识库"
- **查**：写作过程中随时查阅知识库，不用翻回前文找"这个人第 3 章是怎么写的"
- **导出**：一键导出为 DOCX/TXT/PDF/EPUB，交稿直接用

简单说：它是一个**融合了 AI 知识管理的专业写作工作室**。

### 和普通写作工具的区别

| | 传统写作工具 | bitsNovels |
|---|---|---|
| 写小说 | ✅ 能写 | ✅ 能写 |
| 管理角色设定 | ❌ 手写笔记 | ✅ AI 自动识别 + 关系图谱 |
| 伏笔追踪 | ❌ 靠脑子记 | ✅ AI 帮你记、到期提醒 |
| 设定一致性检查 | ❌ 人工校稿 | ✅ AI 自动检测前后矛盾 |
| 导出格式 | TXT 为主 | DOCX/TXT/PDF/EPUB 全覆盖 |

## 视觉风格

bitsNovels 采用了"**复古理科与数字羊皮纸**"的设计语言：

- **羊皮纸暖色调**：背景是泛黄的米白色，不是刺眼的纯白；文字是褪色墨水的深棕黑，不是死黑的纯黑。长时间盯着屏幕不累眼睛。
- **琥珀色点缀**：主色调不是冷冰冰的科技蓝，而是像黄铜笔尖划过羊皮纸的反光，带有一点"蒸汽朋克"的复古感。
- **三栏沉浸布局**：左边章节树、右边知识库、中间编辑器。左右两侧可以收起来，编辑器永远是你目光的绝对中心。

详细设计规范见 [DESIGN_SYSTEM.md](./DESIGN_SYSTEM.md)。

6 个 Epic 覆盖的功能模块：

| Epic | 包含功能举例 |
|---|---|
| **Epic 1 · 项目管理** | 注册登录、项目列表、新建项目、卷章目录管理、写作目标、回收站、归档、导入已有作品 |
| **Epic 2 · 知识库与 Parser** | AI 自动识别角色/地点/道具/势力/伏笔、关系图谱、一致性检查、知识库搜索、世界观设定管理 |
| **Epic 3 · 写作工作台** | 富文本编辑器（撤销/自动保存/查找替换）、专注模式、版本快照、内容批注、写作统计、计时器 |
| **Epic 4 · AI 辅助写作** | AI 续写、AI 润色、AI 扩写/缩写、AI 对话生成、AI 章节大纲建议、AI 名字生成器、写作建议 |

| **Epic 6 · 系统与设置** | 用户资料、界面偏好、快捷键配置、通知中心、存储管理 |

非功能需求（NFR）也已明确定义，包括编辑器性能（输入延迟 ≤ 50ms）、AI 解析性能（单章 ≤ 45s）、可用性目标（99.5%）等。

### V1 不做的事（Scope 声明）

以下功能**明确不在 V1 范围内**，属于 V2 规划：

### 建议开发阶段（参考）

> 以下为建议节奏，非强制承诺。实际进度取决于可用时间。

| 阶段 | 内容 | 预计工作量 |
|---|---|---|
| **Phase 0 · 基建** | 项目骨架（前端框架、UI 组件库、色彩/字体系统）、登录注册、项目列表 | 约 2~3 周 |
| **Phase 1 · 核心写作** | 编辑器核心（Epic 3 写作工作台）、卷章管理（Epic 1 部分） | 约 3~4 周 |
| **Phase 2 · 知识库** | Parser 架构（Epic 2）、知识库面板、关系图谱 | 约 3~4 周 |
| **Phase 3 · AI 功能** | AI 续写/润色/扩写缩写（Epic 4）、一致性检查 | 约 2~3 周 |
| **Phase 4 · 导出备份** | 多格式导出、自动备份、知识库导入导出（Epic 5） | 约 1~2 周 |
| **Phase 5 · 完善** | 通知中心、存储管理、快捷键、偏好设置（Epic 6）、专注模式、版本快照 | 约 2~3 周 |

**全部完成后 ≈ V1 MVP**

### 开发模式参考

- **前端**：建议 React + TypeScript（强类型，适合复杂状态管理）
- **编辑器**：建议基于 Slate 或 ProseMirror 定制（支持富文本+自定义 schema）
- **后端**：建议 Node.js（统一语言栈）或 Go（高性能 API）
- **数据库**：PostgreSQL（结构化数据）+ 向量数据库（AI 知识库检索）
- **AI**：OpenAI GPT API 或等效服务
