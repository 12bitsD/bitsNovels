# CONSTRAINTS · 非功能约束表

> 所有功能和架构决策必须满足以下约束。这些是**隐性炸弹**——不写在文档里，Agent 容易忽略，人容易忘记，但一旦出问题就是大问题。
>
> 约束分为三类：
> - **PERF** — 性能指标（响应时间、吞吐量）
> - **REL** — 可靠性指标（可用性、数据安全）
> - **COMPAT** — 兼容性指标（浏览器支持、版本范围）

------

## PERF · 性能约束

### PERF-01 · 编辑器性能基线

| 场景 | 指标 | 说明 |
|---|---|---|
| 编辑器首次加载（内容渲染） | ≤ 2s（P95） | 含章节内容渲染完成 |
| 章节切换（内容加载 + 面板刷新） | ≤ 1s（P95） | 切换章节的感知速度 |
| 自动保存响应 | ≤ 500ms | 用户无感知，但必须快 |
| 编辑器输入延迟 | ≤ 50ms | 10 万字章节内仍需满足 |
| 撤销/重做响应 | ≤ 100ms | — |

**来源**：UserStory NFR-PERF-01

### PERF-02 · AI 相关性能

| 场景 | 指标 | 说明 |
|---|---|---|
| Parser 单章解析（≤ 5000 字） | ≤ 45s | 异步，不阻塞用户 |
| AI 续写/润色首 token 响应（流式） | ≤ 3s | 用户感知到"开始生成"的时间 |
| 批量解析并行上限 | 5 章 | 超出自动排队 |
| AI 请求超时 | 60s 后重试一次，再失败标记失败 | — |

**来源**：UserStory NFR-PERF-02

### PERF-03 · 搜索与查询

| 场景 | 指标 | 说明 |
|---|---|---|
| 知识库搜索（US-2.9） | ≤ 500ms（P95，1000 条内） | — |
| 章节内查找替换（2 万字内） | ≤ 200ms | — |
| 项目列表加载（50 个项目内） | ≤ 1s | — |

**来源**：UserStory NFR-PERF-03

### PERF-04 · 导出性能

| 场景 | 指标 | 说明 |
|---|---|---|
| 单格式导出（20 万字全书） | ≤ 60s | 异步执行，不阻塞编辑器 |
| 项目完整备份（含快照） | ≤ 120s | 同上 |

**来源**：UserStory NFR-PERF-04

## REL · 可靠性约束

### REL-01 · 可用性目标

| 指标 | 值 |
|---|---|
| 服务可用性目标 | 99.5%（月度） |

**来源**：UserStory NFR-PERF-05

### REL-02 · 数据安全

| 约束 | 说明 |
|---|---|
| 自动保存零丢失 | 网络恢复后自动重试，本地保留未同步草稿 |
| 页面崩溃恢复 | 重新打开后自动恢复到最后一个自动保存的内容 |
| 回收站不可逆删除 | 30 天后自动永久删除，用户主动永久删除需二次确认 |
| 备份覆盖操作不可逆 | US-5.5 "覆盖当前项目"需输入项目名确认 |

**来源**：UserStory NFR-PERF-05、US-1.7、US-5.5

### REL-03 · 会话与安全

| 约束 | 说明 |
|---|---|
| 密码强度 | ≥ 8 位，包含大小写字母和数字 |
| 邮箱验证链接有效期 | 24 小时 |
| 密码重置链接有效期 | 1 小时 |
| 记住我会话 | 30 天 |
| 非记住我会话 | 24 小时 |
| 密码重置后 | 所有已登录设备会话自动失效 |

**来源**：UserStory US-1.1、US-1.4

## COMPAT · 兼容性约束

### COMPAT-01 · 浏览器支持

| 支持范围 | 说明 |
|---|---|
| Chrome / Firefox / Edge | 最新两个大版本 |
| 最小视口宽度 | 1280px（桌面端） |
| 平板端（768px~1279px） | 基础可用布局，不做专项优化 |
| 移动端（<768px） | V1 不支持 |

**来源**：UserStory SCOPE-03

### COMPAT-02 · 文件导入兼容

| 格式 | 限制 | 编码检测 |
|---|---|---|
| TXT | ≤ 20MB | UTF-8 / GBK / GB2312 自动检测 |
| DOCX | ≤ 20MB | — |

**来源**：UserStory US-1.9

### COMPAT-03 · 备份格式兼容性

| 约束 | 说明 |
|---|---|
| 备份 ZIP 向前兼容 | 至少兼容 2 个主版本 |
| 格式损坏检测 | 上传时校验完整性，无效则拒绝 |

**来源**：UserStory US-5.5

### COMPAT-04 · V1 明确不支持的功能（Scope）

以下功能不在 V1 范围内，**任何实现讨论必须先确认是否触碰这些红线**：

- ❌ 跨项目全局搜索
- ❌ 多人协作 / 项目共享 / 权限管理
- ❌ 移动端适配
